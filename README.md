
## 目录结构

```plaintext
web3-engineering-starter-kit/
├── apps/
│   ├── web/               # 主 DApp 前端
│   └── contracts/         # Hardhat 合约工程
├── packages/
│   ├── sdk/               # 按需加载的多链 SDK
│   ├── ui/                # 钱包组件库，ESM 格式发布
│   └── config/            # 多链配置、abi 类型生成
├── .husky/                # 预提交钩子
├── .github/workflows/     # CI / 自动测试
├── docker/                # 本地 dev 容器环境
├── docs/                  # 工程化文档、部署说明
├── package.json/  
├── node_modules/
├── turbo.json/
├── pnpm-lock.yaml/      
└── README.md
```
